# Personal Website

🔗 Live Demo: https://shinjokang.com

## Overview

This is a personal portfolio website built to showcase my technical skills and projects. It presents my development experience, highlights selected recent work, and provides detailed pages for each project.

## Tech Stack

- **Backend:** Python, Django
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, JavaScript
- **Cloud & Deployment:** Microsoft Azure
- **Storage:** Azure Blob Storage
- **Version Control & CI/CD:** GitHub (integrated with Azure for automatic deployment)

## Features

#### Responsive Design

- Fully responsive layout optimized for desktop and mobile devices

#### Project Showcase

- "My Recent Projects" section displaying the latest 3 projects
- "All Projects" page to browse the complete list of projects

#### Project Detail Page

- Displays project title, image, and description
- Includes GitHub repository links for personal projects
- Includes live deployment links for professional (work-related) projects

#### Navigation

- Quick access to GitHub and LinkedIn via the navigation bar

#### Cloud Integration

- Images are stored and served using Azure Blob Storage
- Application is deployed on Azure Web App

#### Automatic Deployment

- GitHub is connected to Azure, enabling automatic deployment on every push

## How It Works

1. Develop locally using Django
2. Commit and push changes to GitHub
3. Azure automatically pulls the latest changes and redeploys the application
4. Images are stored and served via Azure Blob Storage

## Purpose

The goal of this project is to present my skills and projects in a clean, structured, and professional way for recruiters and employers.
